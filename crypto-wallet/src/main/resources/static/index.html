Crypto Wallet API Documentation

Manage your crypto assets, update prices, and analyze performance.

Overview

This API allows you to manage a collection of crypto assets (a wallet) with functionalities to:

  • Add new assets to your wallet.
  • Update asset prices concurrently (using CoinCap API).
  • Retrieve the wallet summary (total value, best/worst asset performance).
  • Fetch all wallet history.

API Endpoints

1. Add Assets to Wallet

URL: POST /api/wallet/add

Description: Add a list of assets to the wallet.

Request Body (JSON):

[
  { "symbol": "BTC", "quantity": 0.12345, "price": 37870.5058 },
  { "symbol": "ETH", "quantity": 4.89532, "price": 2004.9774 }
]

Response:

Asset added successfully

2. Update Prices Concurrently

URL: GET /api/wallet/update

Description: Fetch the latest prices for all wallet assets.

Response:

Prices updated successfully

3. Get Wallet Summary

URL: GET /api/wallet/summary

Description: Retrieve the wallet summary, including total value, best and worst performance.

Response Example:

Total: 20500.50, Best Asset: BTC (5.23%), Worst Asset: ETH (-2.14%)

4. Get Wallet History

URL: GET /api/wallet/history

Description: Retrieve all assets stored in the wallet.

Response Example:

[
  { "id": 1, "symbol": "BTC", "quantity": 0.12345, "price": 37870.5058, "latestPrice": 39000.00 },
  { "id": 2, "symbol": "ETH", "quantity": 4.89532, "price": 2004.9774, "latestPrice": 1980.50 }
]

Testing

You can test this API using tools like Postman or curl.
